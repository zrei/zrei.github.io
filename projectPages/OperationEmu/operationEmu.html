Operation Emu

  • itch.io
  • Github

A top-down dungeon explorer developed for a game design module. The game was developed in Gamemaker and I was the sole developer. I implemented mechanics based on my team's design ideas, incorporating assets from the artist.

This was a valuable experience as I had to build the entire game from the ground up, which allowed me to dabble in areas that I hadn't before and learn about game architecture.
