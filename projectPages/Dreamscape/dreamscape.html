Dreamscape

  • itch.io
  • Gitlab
  • UVENTS Page
  • YouTube

Dreamscape is a parkour game with three levels, developed in Unreal over a semester for a school module. It was the 1st place winner at the 24th SoC Term Project for CS3247.

I was responsible for:

  • Backend systems, such as sound management, level transitions, UI management, save system etc.
  • Hand animation
  • Implementations for the rocket jump
  • Checkpoints, collectibles and respawning
  • Editor tools such as mass texture changes for selected objects to help with the workflow

This was a learning experience as it was my first time touching Unreal, and also working on a larger project for a prolonged period of time outside of my internships. I was able to utilise the lessons I'd learned from my internships to help with project management and ensure we met deadlines with viable prototypes.
